# Svelte

A boilerplate for Svelte, Babel, and PostCSS.
with tailwind

## Getting Started

1. Clone this boilerplate

```bash
$ git clone --depth 1 https://github.com/Timeshiftcorp/svelte-boilerplate <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

2. Install dependencies

```bash
$ yarn install
```

3. Start a local server

```bash
$ yarn start
```
